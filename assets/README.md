# 🎨 Assets

> Images, diagrams, banners, and media files used throughout the repository.

---

## 📁 Structure

```
assets/
├── README.md          ← You are here
├── banner.png         ← Repository banner image
├── diagrams/          ← Architecture and workflow diagrams
├── screenshots/       ← Tool and project screenshots
└── icons/             ← Icons and logos
```

---

## 📋 Naming Conventions

| Type | Convention | Example |
|------|-----------|---------|
| Banners | `banner.png` | `banner.png` |
| Diagrams | `section-name-diagram.png` | `git-workflow-diagram.png` |
| Screenshots | `tool-name-screenshot.png` | `vscode-screenshot.png` |
| Icons | `tool-name-icon.png` | `docker-icon.png` |

---

## 📝 Guidelines

- Use **PNG** for diagrams and screenshots
- Use **SVG** for icons and logos when possible
- Keep file sizes reasonable (< 500KB per image)
- Use descriptive, lowercase filenames with dashes
- Organize files into subdirectories by type
